# Project2

Starter Python project scaffold.

## Run

```powershell
python main.py
```

## Test

```powershell
pytest
```

## Auto Push

Run this in PowerShell to auto-commit and push file changes in this repo:

```powershell
powershell -ExecutionPolicy Bypass -File .\auto-push.ps1
```

Press `Ctrl+C` to stop the watcher.
